# Role: 高级前端 UI/UX 设计师与架构师
# Task: 设计一套“材质报告 AI 自动审核系统”的后台管理系统全局布局

## 1. 风格与技术栈要求
- **风格**：现代、简洁、微工业风/科技感。整体色调以工业蓝、科技灰为主，搭配清晰的成功绿（PASS）和警告红（FAIL）。
- **组件规范**：页面美观、紧凑，杜绝大面积留白，适合后台高频操作场景（推荐类似 Element Plus 或 Tailwind CSS + Shadcn 风格）。
- **响应式**：标准桌面端自适应（1920x1080 优化）。

## 2. 角色与权限菜单结构（不要太复杂）
系统包含两种用户角色，左侧菜单根据角色进行响应式显示或置灰：
1. **管理员 (Admin)**：
   - 仪表盘概览（Dashboard）
   - AI 审核工作台（Core Workspace）
   - 系统用户管理（User Management）
2. **审核员 (Auditor)**：
   - 仪表盘概览（Dashboard）
   - AI 审核工作台（Core Workspace） - *仅核心操作权限*

## 3. 全局布局组件
- **TopBar（顶部栏）**：左侧为系统名称“材质报告 AI 自动审核系统 V1”，右侧为当前用户头像、角色标签（如：管理员）、退出登录按钮。
- **SideBar（侧边栏）**：垂直导航菜单，支持高亮显示当前激活路由。
- **MainContent（主内容区）**：带面包屑导航，卡片式容器承载具体页面。
