# Task: 生成“材质报告 AI 自动审核系统”的登录页面

## 1. 视觉设计
- **卡片式居中布局**：左侧可放置一个工业质感的 AI 视觉识别抽象背景图或系统 Logo，右侧为登录表单。
- 整体风格干净利落，符合企业级 B 端软件标准。

## 2. 表单元素
- **输入框 1**：用户名/邮箱（带用户图标内联修饰）。
- **输入框 2**：密码（带眼睛图标，支持明密文切换）。
- **角色选择（关键）**：提供一个简洁的单选框（Radio Group）或下拉选择框（Select），供演示切换角色：[ 管理员 ] 和 [ 审核员 ]。
- **动作按钮**：一个宽幅的“登 录”主按钮，带加载状态（Loading）和 hover 渐变动效。
- **辅助链接**：勾选框“记住密码”，以及轻量化的“忘记密码”文本。
